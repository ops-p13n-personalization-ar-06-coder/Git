# SHIPD Helper Kit

This folder contains ready-to-use guides, prompts, and checklists for solving Shipd/Mars problems with zero local setup.

Recommended read order:
1. `PLAYBOOK.md`
2. `SOLUTION_PATCH_GUIDE.md`
3. `SUBMISSION_CHECKLIST.md`
4. `DIFF_TROUBLESHOOT.md`

Bonus:
- `CODEX_PROMPTS.md` → copy/paste to Codex
- `COMMAND_SNIPPETS_WIN_GITBASH.md` → quick validation (optional)
- `ELIGIBILITY_CHECKLIST.md` → repo/issue selection
- `FEEDBACK_REPLY_TEMPLATES.md` → answer reviewers fast
- Problem-specific notes go in files like `SCIKIT_LEARN_*.md`
